#🔖 The Coding Lab - Full Featured Blog Website

Welcome to **The Coding Lab**, a modern, responsive blog platform designed for web developers and tech enthusiasts. This project focuses on clean UI/UX design, modular CSS, and a structured layout for articles and user interaction.

---

## 💫 Features

- **Responsive Modern UI**: A clean aesthetic built with custom CSS variables for consistency.
- **Dynamic Article Grid**: A structured layout for browsing multiple blog posts using CSS Grid.
- **Hero Section**: Featured article highlighting important content with high-impact visuals.
- **Custom Pagination**: Easy navigation through different pages of the blog.
- **Contact Interface**: A sleek, user-friendly contact form with a professional slanted-edge design.
- **Modular Styles**: CSS is split into specific files (`style.css`, `article.css`, `contact.css`, `footer.css`) for better maintainability.

---

## ⚔️ Built With

- **HTML5**: Semantic structure for better SEO and accessibility.
- **CSS3**: Custom properties (Variables), Flexbox, and CSS Grid.
- **Font Awesome**: Professional iconography for UI elements.
- **Google Fonts**: Using "Poppins" for body text and "Jersey 25" for specialized headings.
